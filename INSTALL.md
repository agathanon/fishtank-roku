## Installation

### 1. Enable Developer Mode on your Roku

Using your Roku remote, go to **Settings → System → About**, then press:

**Home Home Home Up Up Right Left Right Left Right**

A developer settings page will appear. Enable the installer and set a password when prompted. Note your Roku's IP address shown on this screen (something like `192.168.1.100`).

### 2. Download the latest release

Go to the [Releases](https://github.com/agathanon/fishtank-roku/releases) page and download the `.zip` file from the latest version. Do not unzip it.

### 3. Open the Roku Installer

On a computer connected to the same network as your Roku, open a web browser and go to:

```
http://YOUR_ROKU_IP
```

Replace `YOUR_ROKU_IP` with the IP address from step 1 (e.g., `http://192.168.1.100`). When prompted, log in with the username `rokudev` and the password you set in step 1.

### 4. Upload and install

On the installer page, click **Upload** and select the `.zip` file you downloaded. Then click **Install**. The app will launch automatically on your Roku.

### 5. Log in

Enter your Fishtank.live email and password using the on-screen keyboard. You only need to do this once - the app will remember your session.
